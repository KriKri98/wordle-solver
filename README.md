This project helps on the famous wordle.
If you are unsure, which words are availible you can use this tool to see which words are availiable.

Simply install Python and copy the main.py and the valid-wordle-words.txt.
Upon starting the programm, you need to input a 5-letter word.
After that, you will be asked to assign every letter a color:
g for green: This letter is in the correct spot
y for yellow: This letter is in the word, but in the wrong spot
b for black: This letter is not in the word

You will get an output of all remaining words and a list of overall letters left with their respective amount.

You can exit the program while typing "exit" into the word input.
The program will also end, if there are no words left.
